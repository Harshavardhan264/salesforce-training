# Salesforce Basics 🚀

## 1. What is Salesforce Platform? ☁️

Salesforce Platform is a cloud-based platform used to build business applications and manage customer data efficiently.

### Features ✨
- CRM support 🤝
- Cloud storage ☁️
- Automation ⚡
- Reports & Dashboards 📊

---

# 2. Important Terms 📚

## App 📱
An App is a collection of tabs, objects, and tools designed for a specific business purpose.

**Example:** Sales App, Library App

---

## Object 📦
An Object stores data in Salesforce, similar to a database table.

### Types:
- Standard Objects
- Custom Objects

**Example:** Student, Book, Employee

---

## Tab 🗂️
A Tab helps users access objects and records easily from the interface.

**Example:** Student Tab, Dashboard Tab

---

# 3. Configuration vs Coding ⚖️

| Configuration 🔧 | Coding 💻 |
|------------------|-----------|
| Drag-and-drop tools | Uses programming |
| No coding required | Requires coding skills |
| Faster development | More customization |

# 4. System Design 🏗️

## Example: Library Management App 📚

### App:
- Library Management App

### Objects:
- Book 📘
- Student 👨‍🎓
- Issue Record 📝

### User Interaction 👥
1. Admin adds books ➕
